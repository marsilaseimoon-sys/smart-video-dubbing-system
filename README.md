# 🎥 AI Video Dubbing System

This project is an AI-powered video dubbing application that automatically converts speech from a video into text and generates a new voice-over using advanced Text-to-Speech (TTS) technology.

## 🚀 Features
- 🎙 Speech-to-Text transcription (using Whisper)
- 🌍 Multi-language translation support
- 🔊 AI voice generation (Text-to-Speech)
- 🎬 Audio-video synchronization using FFmpeg
- ⚡ Fast and automated dubbing process

## 🛠 Technologies Used
- Python
- OpenAI Whisper (Speech Recognition)
- Text-to-Speech (TTS)
- FFmpeg
- Machine Learning

## 🎯 Purpose
This project aims to simplify video localization by automating the dubbing process, making content accessible to a global audience.

## 📌 Future Improvements
- Real-time dubbing

- Multiple voice styles
